This roadmap will guide you through the essential topics to learn as a CS student from scratch.

1. Programming Fundamentals

  • Choose a language: Python or C
  • Variables, Control Flow, Loops
  • Functions, Recursion
  • Basic Data Structures (Arrays, Lists, Stacks, Queues)

2. Data Structures & Algorithms

  • Linked Lists, Trees, Graphs
  • Sorting & Searching Algorithms
  • Time and Space Complexity

3. Computer Architecture & OS

  • Binary, Logic Gates
  • Memory Management, Processes
  • File Systems, Scheduling

4. Databases

  • SQL Basics
  • Database Design, Normalization

5. Networking

  • TCP/IP, DNS, HTTP
  • OSI Model

6. Software Development

  • Version Control (Git)
  • Debugging, Testing
  • Projects and Collaboration

7. Extras

  • Web Development Basics
  • Linux Command Line
  • Problem Solving (Competitive Programming)
